Политика конфиденциальности приложения «Профориентация»

Дата: [13.07.2026]

1. Общие положения
Приложение «Профориентация» (ru.lf.proforientaciya) предоставляется бесплатно.
Используя приложение, вы соглашаетесь с этой политикой.

2. Какие данные собираются
Приложение НЕ собирает и НЕ передаёт персональные данные на серверы разработчика.
Все данные (имя пользователя, ответы на тесты, результаты, настройки темы и режима ответов)
хранятся только локально на вашем устройстве.

3. Интернет
Приложение может открывать внешние ссылки по вашему действию:
- страница доната (ЮMoney)
- Google Play (оценка приложения)
- окно «Поделиться» (мессенджеры и соцсети по вашему выбору)
Эти действия выполняются только когда вы нажимаете соответствующие кнопки.

4. Дети
Приложение не предназначено для сбора данных детей. Родители могут использовать
приложение для профориентации ребёнка на своём устройстве.

5. Удаление данных
Удалите приложение с устройства — все локальные данные будут удалены вместе с ним.
Также можно сбросить данные в настройках Android: Приложения → Профориентация → Очистить данные.

6. Изменения
Политика может обновляться. Актуальная версия доступна по этой ссылке.
