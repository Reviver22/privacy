# Политика конфиденциальности приложения «Профориентация»
**Дата:** 13.07.2026

---

### 1. Общие положения
Приложение «Профориентация» (`ru.lf.proforientaciya`) предоставляется бесплатно. Используя приложение, вы соглашаетесь с этой политикой.

### 2. Какие данные собираются
Приложение **НЕ собирает** и **НЕ передает** персональные данные на серверы разработчика. 

Все данные хранятся **только локально** на вашем устройстве:
* Имя пользователя
* Ответы на тесты и результаты
* Настройки темы и режима ответов

### 3. Интернет
Приложение может открывать внешние ссылки по вашему действию:
* Страница доната (ЮMoney)
* Google Play (оценка приложения)
* Окно «Поделиться» (мессенджеры и соцсети по вашему выбору)

Эти действия выполняются *только тогда*, когда вы нажимаете соответствующие кнопки.

### 4. Дети
Приложение не предназначено для сбора данных детей. Родители могут использовать приложение для профориентации ребенка на своём устройстве.

### 5. Удаление данных
Удалите приложение с устройства — все локальные данные будут удалены вместе с ним. Также можно сбросить данные в настройках Android: *Приложения → Профориентация → Очистить данные*.

### 6. Изменения
Политика может обновляться. Актуальная версия доступна по этой ссылке.
